常见问题 / FAQ

录音 / Recording

  • 如何录音？打开应用 → 进入“录音”页面 → 点击录制按钮。
  • 录音时长有限制吗？是的，默认限制为 15 秒（可在界面查看）。
  • 录音文件存放在哪里？录音保存在 App 的本地目录（Library/Sounds），不会上传到服务器。

导入与播放 / Import & Playback

  • 支持导入哪些格式？常见音频格式（如 m4a、mp3）通常可用，具体依设备和格式而异。
  • 导入后如何设为铃声？导入文件后在文件列表中选择“设为铃声”。
  • 播放问题：若无法播放，请检查文件是否损坏或格式受限。

内购 / In-App Purchase

  • 如何购买 VIP？在“购买”页点击购买按钮，使用 Apple ID 完成支付。
  • 如何恢复购买？设置或购买页点击“恢复购买”。购买与恢复均通过 App Store 处理。
  • 遇到购买问题？请使用邮件联系我们并附上你的 Apple ID 邮箱与购买时间（若可用）。

邮件反馈 / Mail Feedback

  • 若无法打开邮件应用，请选择“复制邮箱地址”，然后手动发送邮件或在浏览器打开 webmail。
  • 邮件示例内容建议：App 版本、iOS 版本、问题描述、复现步骤、截图（如有）。

故障排查 / Troubleshooting

常见快速修复步骤：

  1. 重启 App（双击 Home 或上划强制关闭后重新打开）。
  2. 确保已授予麦克风权限：设置 → EchoClock → 麦克风（Microphone）。
  3. 如果导入文件无法播放，尝试在外部转换为 m4a 或 mp3 再导入。
  4. 若内购失败，请确认 App Store 登录状态并使用真实设备测试（非模拟器）。
  5. 删除 App 并重新安装（重装会删除本地数据，请谨慎）。
